# touhouworldcup/thpracからRUEEE/thpracで追加された機能

[RUEEE版thpracのREADME](https://github.com/RUEEE/thprac/blob/master_2/README.md)から抜粋・翻訳しています。抜けや誤訳等があるかも。

## 全般
- ゲーム内情報表示機能 (ゲーム起動→Backspaceキー入力→「ゲーム内情報」)
- keng tools (打開支援ツール)
- 反射神経テスト
- 入力テストツール
- SOCD機能
- キーボードAPI設定
- キー割当機能
- フォント修復&カスタムフォント対応
- ゲームルーレット
- csvファイルからルーレットを作成
- サイコロ (6面体/12面体)
- ゲーム起動時間記録、ゲーム起動時間超過警告機能
- バックグラウンドで動作しているゲームを閉じる機能
- Windowsキー無効化オプション
- キー入力表示オプション
- ジョイスティック無効化オプション
- モニターのリフレッシュレートのロックを解除するオプション
- ゲーム開始前にウィンドウサイズを変更可能に
- ボスの下移動制御オプション
- 【紅～永】弾幕当たり判定表示オプション (Practice/Replay限定)
- 【紅,風,星,紺】視界暗転オプション
- 【妖,永】チェックサムを無効化 (中国語版のみ影響？)
- 【妖,永,風】F12メニューから1クリックでゲームオーバー (高速でリプレイを保存可能)
- 【永～鬼】Master表記無効化オプション (スペルカード取得率を完全に表示)
- 【地,星,神】F10キー無効化オプション (F10でフルスクリーンに切り替えられないように)
