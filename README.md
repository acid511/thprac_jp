## 使い方

基本はRUEEE版thpracと同じです。

### touhouworldcup/thpracからRUEEE/thpracで追加された機能

[FEATURES.md](https://github.com/acid511/thprac_jp/blob/main/FEATURES.md)にまとめています。

RUEEE/thpracからacid511/thprac_jpで追加された機能は日本語化を除けばほぼありません。

### 打開支援ツールについて

keng tools (打開支援ツール)は、特定の箇所で被弾した回数を記録し突破率を計算することで、攻略の手助けをするツールです。 (主にLNNFS向け)

#### セットアップ

1. 「課題を追加」
2. 名前、日付、説明を入力後、プリセットを選択 → 「適用する」

![セットアップ](https://github.com/user-attachments/assets/6871aedc-1f94-4608-b737-caac4b713863)

#### 記録の追加

1. 「記録を追加」
2. 記録名、日付、コメントを入力後、被弾箇所を選択 (同じ場所で複数回被弾しても1回分としてカウントされるので、コメントに詳細を書いておくと良いかも)
3. 下の方にスクロール → 「適用する」

![記録の追加](https://github.com/user-attachments/assets/e0e769a9-3b72-4d11-8185-acd1615da91d)

#### 突破率の確認

1. 「詳細」
2. 突破率や突破に必要な所要回数を確認可能

![確認](https://github.com/user-attachments/assets/9b040ace-b441-4666-984f-3d952c852779)
